クロスフォース採用、「三代目」クロステージ登場。

デビューから十年、全てを一新した「三代目」クロステージが遂に登場する。初中級アングラーが気軽に本格的なスペックを体感できる、まさにタックル革命をもたらした初代のコンセプトはそのままに、年々刻々と変化するフィールドコンディションとフィッシングメソッドに対応するべくマテリアルから再検証。
最新テクノロジー「クロスフォース製法」も採用し、新たに開発されたブランクが、より快適に、よりシャープになってアングラーを全力サポート。握りやすい細身のグリップ形状が、快適なゲームを約束する。コストパフォーマンスに優れたスタンダードロッドの代名詞クロステージは今、「三代目」の時代に突入した。

●CRX-T732L TUBULAR model
コントロール性能に優れ、ルアーをシモリ際に通したい状況などで活躍するモデル。

●CRX-T762L TUBULAR model
ワームからミノーやシンキングペンシル、マイクロジグまで幅広く対応するモデル。

●CRX-T792L TUBULAR model
シンキングペンシルやフロートリグの大遠投が可能。マイクロジグとの相性も抜群。

●CRX-T762ML TUBULAR model
マイクロジグの操作性に優れた張りのあるブランクスで、デカメバルを攻略。

●CRX-S702UL SOLID TIP model
アキュラシー性能に優れたモデル。 小規模な漁港のラン＆ガンに最適。

●CRX-S732UL SOLID TIP model
一般的な漁港で使い勝手の良いモデル。 常夜灯周辺など定番ポイントに最適。

●CRX-S762UL SOLID TIP model
いつでもどこでも活躍するレングス設定のバーサタイルソリッドティップモデル。

●CRX-S792UL SOLID TIP model
軽量リグをロングキャストして、スレていない良型メバルを狙うためのモデル。
