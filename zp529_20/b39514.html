ハッセルブラッド RELEASE CODE X CP.HB.00000242.01 CP.HB.00000242.01

ハッセルブラッドのレリーズコードXは、リモートシャッターを可能にし、振動を排除するための重要なツールとなります。耐久性のある90cmの布引ケーブルをX1D-50cのマイク入力ポートに接続し、リモートレリーズのボタンを押すことで振動を最小限に抑えることができます。カメラ側でオートフォーカスが有効になっていると、ボタンを1回押すとオートフォーカスが開始し、続いてシャッターが切れます。 連写モードでボタンを押し続けると、複数枚の撮影が可能です。一方、カメラをマニュアルフォーカスモードに設定すると、遅延なくすぐにシャッターが切れます。
